# Google HashCode
Implementations for 2018, 2019 and 2020 hashcode qualification rounds.

## Qualification 2018
- Language used: *Java*
- Points: *14,896,001*
- Usage: ```$ ./script.sh```

## Qualification 2019
- Language used: *Python*
- Points: *207,009*
- Usage: ```$ python <in.file>```

## Qualification 2020
- Language used: *Python*
- Points: *12,930,045*
- Usage: Jupyter Notebook
